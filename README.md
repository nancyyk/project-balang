# BALANG - Barang Hilang & Ditemukan

BALANG adalah website pelaporan barang hilang dan barang ditemukan.

## Fitur
- Login & Register
- Lapor Barang Hilang
- Lihat History Laporan
- Profile User
- Dashboard User

## Teknologi
- PHP Native
- Tailwind CSS
- MySQL
- MVC Structure

## Cara Menjalankan
1. Clone repository
2. Pindahkan ke folder laragon/www
3. Jalankan Laragon
